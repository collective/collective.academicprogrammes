Introduction
============

An add-on for listing academic programmes.
Each programme consists of various courses. Some programmes share the same courses, so the 
goal was to provide an interface that supports easily associating a course with one or more
programmes.

Some courses are prerequisites for other courses and this can be configured in the system as well.

The product was originally created use by the http://CARIMAC.com website so that they could 
Easily manage their academic programmes.
